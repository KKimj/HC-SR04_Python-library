# HC-SR04_Python-library
HC-SR04_Python-library for PyPi

## Install
```
$ pip install HC-SR04
```

## Build
```
$ python3 -m build
```

## Local test
```
$ pip install -e .
```

### Build and Local test
```
$ python3 -m build && pip install -e . && python
```

## Release
```
$ python -m twine upload dist/*
```
